NASA Capture

By Caleb Vaccaro

How To:

  1. Use the "Incoming Data" Section to observe different images from different NASA APIs (libraries).
  2. The "Notes" Section is the area where you jot down notes about the information you can make out from the images.
  3. If you want to add a new NoteSet (Title, Notes), click "NoteSet" and Press "Add" when you are done with initial data.
  4. When you are completed using NASA Capture be sure to save your data with the "Save Data" button.
  5. The rest is up to you, what do YOU see in the observation information?

Reasons for Development:

Started off as a project for my Rich Media Web App Dev I that was supposed to capture two APIs, with firebase connections. This was the main driving force for completing the app for a while, but then I realized how impactful this type of project can be for many. Anyone who is intersted in observing the data that is captured by astromonomers today, can help them understand space and see how vast space really is. There is a ton of data that is captured from camera's, and instruments from NASA and other space departments. Who's to say they didn't looked over something important in the data. No reason why the public can't help pitch in.

Future Ideas/Development

When I was first designing the app, I wanted Notes and Datasets which would be used to capture data. I took out the datasets which were data tables of any type the user wanted to write down. My professor and others, were a bit confused on the application itself and what to do inside the application. So, I reworked on re-directing attention to the API content and soley reflect on that observation data. Looking forward to the future, there is contionous use for NASA Capture and can evolve with new technology such as the launch of the James Webb telescope into space. I am looking forward to a great API to query from.
